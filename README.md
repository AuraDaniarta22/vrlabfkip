# Tur Virtual Laboratorium FKIP (PMIPA)

Selamat datang di repositori **Tur Virtual Laboratorium FKIP (PMIPA)**! 

Proyek ini adalah simulasi Tur Virtual Interaktif berbasis web (WebVR) yang memungkinkan pengunjung untuk menjelajahi lingkungan Laboratorium FKIP (Fakultas Keguruan dan Ilmu Pendidikan) Program Studi Pendidikan MIPA secara imersif (360 derajat).

## Fitur Utama

- **Navigasi 360 Derajat**: Jelajahi setiap sudut laboratorium FKIP PMIPA seolah-olah Anda berada langsung di sana.
- **Informasi Fasilitas Ruangan (Pop-up Modal)**: Menampilkan detail fasilitas dan alat-alat laboratorium melalui antarmuka *pop-up* responsif yang dinamis.
- **Dukungan Fullscreen Penuh**: Dirancang secara khusus untuk bekerja secara optimal dalam mode Layar Penuh (*Fullscreen*) tanpa kehilangan antarmuka penting.
- **Mini-Map**: Peta interaktif yang memudahkan perpindahan antar titik lokasi di dalam laboratorium lantai 1 dan lantai 2.
- **Multi-Device**: Telah dioptimalkan untuk performa maksimal baik saat diakses melalui *smartphone* maupun perangkat *desktop*.

## Teknologi yang Digunakan

Tur virtual ini di-*generate* menggunakan teknologi **3DVista Virtual Tour Pro**, kemudian dikustomisasi lebih lanjut menggunakan skrip **JavaScript (Vanilla JS)** dan **CSS** untuk menyempurnakan performa, UI/UX (termasuk stabilitas z-index), dan sistem modul interaktifnya (seperti perbaikan *bug fullscreen browser* pada elemen kanvas WebGL).
